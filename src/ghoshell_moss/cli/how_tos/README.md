# MOSS How-To 知识库

这里存放 MOSS 项目的操作指南。每个文档是一个 how-to，解释如何完成一个具体任务。

## 如何使用

```bash
# 列出所有 how-to
moss how-to list

# 查看一个
moss how-to read how-to-make-how-to.md

# 问 AI (需要配置环境变量)
moss how-to ask "怎么创建一个新的 how-to 文档?"
```

## 如何贡献

写一个新的 .md 文件放在这个目录下。建议包含 YAML frontmatter (title + description)。README.md 是目录自身的说明。
