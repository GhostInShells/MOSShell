# Existence

Atom 的存在状态, 一种极简的技术实现.

基本的技术原理是: 

1. Atom 运行时创建日志更新任务. 
2. 旁路运行时根据 session / conversation 进行摘要, 记录日记. 
3. 以周为单位做周的摘要. 
4. 以月为单位, 做月的滚动摘要. 
5. 以年为单位, 做年的摘要. 
6. 以 年-月-周-日 的压缩机制, 滚动更新 existence 的描述. 

最重要的是滚动机制 (节省 token) 和生成摘要的 prompt 机制. 

生成的链路机制甚至可以考虑用 Claude Agent 的文件机制实现. 甚至考虑直接用  Claude Agent 在文件里创建.
